# AI-Driven Hospital Readmission Risk Prediction for Diabetic Patients

## Project Overview
This project builds an AI-driven analytics solution to predict whether diabetic patients are likely to be readmitted to the hospital within 30 days. The goal is to support hospital care teams by identifying patients who may need additional follow-up, medication review, discharge education, or care coordination.

## Dataset
The project uses the Diabetes 130-US Hospitals for Years 1999–2008 dataset from the UCI Machine Learning Repository.

Dataset link: https://archive.ics.uci.edu/dataset/296/diabetes+130-us+hospitals+for+years+1999-2008

## Research Question
Can patient demographics, hospital utilization history, diagnoses, medications, and treatment information be used to predict whether a diabetic patient will be readmitted within 30 days?
